Emulação de Cabo Link via Wi-fi para trocas/Batalhas

1° - BIOS no emulador

Para se conectar como Server/Client é preciso ter o arquivo BIOS (LINK NO FINAL DO TUTORIAL) no MyBoy ou PizzaBoy.

Passo a passo de ativação da BIOS:
1° passo - Com o jogo aberto, vá em configurações
2° passo - Link Remote
3° passo - Wi-fi Client
4° passo - Insira o IP PUBLIC do jogador Server e a porta 6374.

2° - Client Server

Para conectar como Wi-fi/Client é preciso inserir o IP PUBLIC do jogador Server

Passo a Passo de conexão do jogador client:
1° passo - Com o jogo aberto, vá em configurações
2° passo - Link Remote
3° passo - Wi-fi Client
4° passo - Insira o IP PUBLIC do jogador Server e a porta 6374.

OBS: Se a ROM for diferente, exemplo: Fire Red vs Emerald aparecerá uma mensagem pedindo para inserir a ROM oposta da sua, na opção LOCATE GAME coloque a ROM e segue normal.

Todas as ROMS de GBA se conectam:
Fire Red Leaf Green Sapphire Ruby Emerald

Aplicativo - Port Forwarder

Para conectar como Wi-fi Server é preciso o aplicativo Apk Port Forwarder

1° passo - Ao iniciar o APK na primeira tela clique em
2° passo - Na segunda tela insira qualquer nome em Name*
3° passo - Em Public Port* e Private Port coloque sempre 6374
4° passo - Clique em Add Port Forwarding
5° passo - na ultima tela aparece em Public IP e Private IP, mande só Public IP e a porta 6374 para o jogador Client, aguarde ele conectar e pronto.
