Emulação de Cabo Link via Wi-fi para trocas/Batalhas

1° - BIOS no emulador

Para se conectar como Server ou Client, é necessário ter o arquivo BIOS onfigurado no MyBoy ou PizzaBoy.

Passo a passo de ativação da BIOS:
1° passo - Abra o aplicativo e vá para Configurações (Settings).
2° passo - Selecione a opção Avançado (Advanced)
3° passo - Ative as opções Use BIOS e Boot BIOS
4° passo - Clique em BIOS File e selecione o arquivo BIOS que você baixou no seu celular.

2° - Aplicativo Port Forwarder (Jogador Server)

Para se conectar como Wi-Fi Server, é necessário utilizar o aplicativo Port Forwarder APK

1° passo - Ao abrir o aplicativo, na primeira tela, clique em .
2° passo - Na segunda tela, insira qualquer nome no campo Name.
3° passo - Nos campos Public Port e Private Port, sempre coloque 6374, após preencher clique em Add Port Forwarding
4° passo - Na última tela, aparecerão os campos Public IP address e Private IP address. Envie apenas o Public IP address e a porta 6374 para o jogador Client.

3° - Conexão Jogador Client

Para conectar como Wi-fi/Client é preciso inserir o Public IP address do jogador Server

Passo a Passo de conexão do jogador client:
1° passo - Com o jogo aberto, vá até configurações(Settings).
2° passo - Selecione a opção Link Remote e escolha Wi-Fi Client
3° passo - Insira o Public IP address do jogador Server e a porta 6374.

OBS: Se as ROMs forem diferentes, por exemplo:Fire Red vs Emerald aparecerá uma mensagem pedindo para inserir a ROM do jogador server. No campo Locate Game, basta selecionar a ROM correta e seguir normalmente.

Todas as ROMs de GBA se conectam:
Fire Red Leaf Green Sapphire Ruby Emerald
